Name

The name of the event is Electromagnetic Field, which should always be written out in full initially, where possible. It should be abbreviated to EMF or EMF{{ event_year }} subsequently.

The abbreviations EMF Camp or EMFCamp should never be used in text when referring to the event. The abbreviation emfcamp may be used only in social media usernames and similar situations, where EMF is unavailable.
