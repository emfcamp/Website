CfP Process Help

This covers technical notes on how the CfP system works. If you're after guidance on how the CfP process works as a submitter you want the guidance page.

It mostly covers the process for submitted talks. Installations, workshops, youth workshops etc. etc. are organised by the relevant teams and are reviewed by them.

The process (for talks)

  1. Submitted
  2. Content check
  3. Anonimisation & reviewability check
  4. Reviewed
  5. Accepted/rejected/added to next round
  6. Accepted talks scheduled

How to review

The aim of this review process is to select the most interesting talks and workshops in as an un-biased a way as we can. To that end all proposals are anonymised and we do blind selection of the best proposals based on how you vote.

Proposals that are ready for you to review can be seen in the review page.

It is split into two lists, 'requires review' and 'previously reviewed'.

The order of proposals in the 'requires review' list is randomised to (hopefully) reduce first/last biasing. The order shouldn't change unless you delete your cookie.

At the bottom is a button 'show some different proposals' which will re-randomise the proposals in your 'requires review' list.

You'll see proposals in blocks of 30. You don't have to review everything but please aim to review at least 1 block of 30.

For each un-reviewed proposal you have several options:

  • Vote (the three green buttons). The options are:
    1. something you don't think we should host that you wouldn't go to.
    2. something interesting but not amazing that you might go to.
    3. something amazing that you would definitely go to
  • If you think you can identify the submitter please write who you think it is in the message box and click this button. This proposal will no longer be shown to you.
  • - if something in the proposal is unclear you can request clarification by filling in the message box and clicking the button.

While you're reviewing you'll be auto-advanced through this list so it's recommended to start from the top but not essential.

Changing your mind

If you want to change your mind you can select previous proposal from "previously reviewed" and click "I'd like to change my response".

How to anonymise

There are two key things we want to know from this stage:

  1. Is there any identifying information?
  2. Does the description (once identifying information is removed) seem to give enough information to know what the talk will be about?

The most common identifying information are:

  • short bios "My name is Jane and I'm...."
  • social media links (e.g. Twitter, LinkedIn)
  • youTube links to previous versions of the talk

If you can, feel free to delete these bits but if it'll require re-writing (and especially if it means the description stops giving enough information to answer question 2, above) click the and we'll help the submitter improve it. {% endblock %}
